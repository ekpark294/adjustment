# 001 한입정산 변경 사항

- 메뉴별 분배 내역의 비고는 참여자 등록 순서대로 표시한다.
- `누가 먹었나요?`의 개별 수량 입력은 `0.5개` 같은 소수 수량을 허용한다.
- 정산 결과 보기의 메뉴별 분배 내역에서 개별 수량 메뉴는 가격 칸에 개당 가격이 아닌 총 가격을 표시한다.
- 개별 수량 메뉴의 분배 금액은 `개당 ₩O,OOO` 형식으로 표시한다.
- 인원별 정산 금액에서 특정 인원을 클릭하면 메뉴별 분배 금액 표에서 해당 인원이 포함된 메뉴 행에 색상을 표시하고, 비고의 해당 인원 이름을 굵게 표시한다.
- Google 광고 심사를 고려해 사용 안내, 개인정보처리방침, 문의 링크를 제공한다.
- 개인정보처리방침은 Google 광고 쿠키, 제3자 광고 기술, 브라우저 localStorage 사용 내용을 포함한다.
